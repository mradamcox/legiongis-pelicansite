background

Arches (www.archesproject.org) is a robust web-based inventory platform designed specifically for cultural heritage resources. Sponsored by the Getty Conservation Institute and World Monuments Fund, Arches is open source software developed by Farallon Geographics. Legion GIS has been fortunate to work closely with all of these organizations to help with support on the Arches user forum, co-present at conferences in the US and abroad, and participate in further development of the software. Through all of these activities we've gained extensive experience with the platform, and will be a great resource to facilitate your organization's interest in Arches.

demos

Want to try Arches out? Go to any of the Legion-hosted demo sites below, and sign in with username = guest and password = guest. These three apps are examples of what Legion GIS is prepared to create or customize for your organization.

AFRH-IRIS app

We worked together with PRESERVE/scapes to create this Arches v3 app for the Armed Forces Retirement Home in Washington DC. We incorporated many historic maps (read about that here), created a multi-faceted permissions system, and reworked the database system to assist with site management and development project planning. visit the repo

Cane River Historic Inventory & Map

Our first work with Arches was to create this app for the Cane River National Heritage Area, in Natchitoches Parish, Louisiana. We created many new map layers (from Sanborn Maps© to high resolution elevation reliefs) to enhance the map view, and migrated many existing datasets to the new installation, including an entire historic cemetery. visit the repo

Arches-HIP

This is a demo version of the Arches Heritage Inventory Package, the default Arches v3 app. Arches-HIP was developed by Farallon Geographics in conjunction with the creation of HistoricPlacesLA the first large-scale Arches v3 implementation. visit the repo

hosting

Even though Arches is open source software (free to use), it still needs to be hosted on a server somewhere, so it is accessible through the internet. Along with data migration, component customization, and final installation, we can take care of hosting for you as well. We use Amazon Web Services for all of our web hosting. This is a reliable and cost effective service that lends itself perfectly to the architecture of the Arches platform. Please get in touch with us to discuss prices for all of our Arches-related services.

v3 utilities

During Arches v3 development, our staff has created some shortcuts here and there that we are happy to share. Feel free to download or contribute.

  • arc2arches ArcGIS toolbox is designed to convert shapefiles or file geodatabase feature classes into the native Arches upload format.
  • arches3_utils is a little set of python scripts to aid in the creation of auxiliary resource-graph related files (authority documents, etc.).
